# 推し活トークテーマアプリ 💖

推し活に特化したトークテーマを提供するWebアプリケーションです。

## ✨ 特徴

- **7つのカテゴリ**: 共通、アイドル、配信者、俳優、声優、キャラクター、スポーツ選手
- **300問以上**: 各カテゴリに豊富な質問を用意
- **ガチャ演出**: 楽しいカプセルアニメーション
- **カラフルなデザイン**: 推しのメンバーカラーをイメージした虹色UI
- **PWA対応**: オフラインでも使えて、ホーム画面に追加可能
- **レスポンシブ**: スマートフォンに最適化
- **検索エンジンブロック**: URLを知っている人だけがアクセス可能（個人利用向け）
