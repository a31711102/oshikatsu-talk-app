# 推し活トークテーマアプリ 💖

推し活に特化したトークテーマを提供するWebアプリケーションです。

## ✨ 特徴

- **7つのカテゴリ**: 共通、アイドル、配信者、俳優、声優、キャラクター、スポーツ選手
- **300問以上**: 各カテゴリに豊富な質問を用意
- **ガチャ演出**: 楽しいカプセルアニメーション
- **カラフルなデザイン**: 推しのメンバーカラーをイメージした虹色UI
- **PWA対応**: オフラインでも使えて、ホーム画面に追加可能
- **レスポンシブ**: スマートフォンに最適化
- **検索エンジンブロック**: URLを知っている人だけがアクセス可能（個人利用向け）

## 🚀 使い方

### ローカルで起動

1. **HTTPサーバーを起動**
   
   Pythonを使う場合：
   ```bash
   cd oshikatsu-talk-app
   python -m http.server 8000
   ```
   
   Node.jsを使う場合：
   ```bash
   cd oshikatsu-talk-app
   npx http-server -p 8000
   ```

2. **ブラウザでアクセス**
   
   `http://localhost:8000` を開く

3. **スマートフォンで確認**
   
   同じWi-Fiネットワークに接続している場合、PCのIPアドレスを使ってアクセス可能
   例: `http://192.168.x.x:8000`
   
   > **⚠️ スマートフォンからアクセスできない場合**
   > 
   > Windowsファイアウォールやネットワーク設定の問題で、スマートフォンから
   > ローカルサーバーにアクセスできない場合があります。
   > 
   > **解決策**: GitHub Pagesにデプロイすれば、どのデバイスからでも
   > 確実にアクセスできます！詳しくは [DEPLOY.md](DEPLOY.md) を参照。

### PWAとしてインストール

- **スマートフォン（iOS Safari）**: 「共有」→「ホーム画面に追加」
- **スマートフォン（Android Chrome）**: メニュー→「ホーム画面に追加」
- **PC（Chrome/Edge）**: アドレスバーのインストールアイコンをクリック

## 📝 質問の追加方法

`data/questions.json` ファイルを編集します。

### 既存カテゴリに質問を追加

```json
{
  "questions": {
    "idol": [
      "既存の質問",
      "新しい質問を追加"  // ←ここに追加
    ]
  }
}
```

### 新しいカテゴリを追加

1. カテゴリ情報を追加：
```json
{
  "categories": [
    {
      "id": "new_category",
      "name": "新カテゴリ",
      "color": "#FF1234",
      "icon": "🎨"
    }
  ]
}
```

2. 質問を追加：
```json
{
  "questions": {
    "new_category": [
      "質問1",
      "質問2"
    ]
  }
}
```

## 🎨 デザインのカスタマイズ

`css/style.css` の `:root` セクションでカラーパレットを変更できます：

```css
:root {
    --color-primary: #FF6B9D;  /* メインカラー */
    --bg-light: #FFF5F7;       /* 背景色 */
    /* その他の色設定 */
}
```

## 🌐 デプロイ

> **💡 推奨**: スマートフォンから確実にアクセスするには、GitHub Pagesへの
> デプロイがおすすめです！無料で簡単に公開できます。

詳しい手順は **[DEPLOY.md](DEPLOY.md)** をご覧ください。

### GitHub Pagesにデプロイ

1. GitHubリポジトリを作成
2. ファイルをプッシュ
3. Settings → Pages → Source: main branch → Save

### Netlifyにデプロイ

1. [Netlify](https://www.netlify.com/)にサインイン
2. フォルダをドラッグ&ドロップ
3. 自動デプロイ完了

### Vercelにデプロイ

1. [Vercel](https://vercel.com/)にサインイン
2. プロジェクトをインポート
3. デプロイ

## 🔒 プライバシー保護について

このアプリは**個人利用向け**に設定されています：

### 現在の設定

- ✅ **検索エンジンブロック**: Google等の検索結果に表示されない
- ✅ **URL直接アクセス**: URLを知っている人だけがアクセス可能
- ✅ **友達とシェア**: URLを友達に送って一緒に使える

### 実装方法

- `robots.txt`: すべての検索エンジンクローラーをブロック
- `<meta>` タグ: noindex, nofollow 設定

**検索では見つからないので、安心して使えます！**
